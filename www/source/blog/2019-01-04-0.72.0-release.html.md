Habitat 0.72.0 Release Notes

We are happy to announce the release of Chef Habitat v0.72.0. We have a new features, as well
as bug fixes so please read on for all the details. If you just want the binaries, head on over to
[Install Chef Habitat](https://www.habitat.sh/docs/using-habitat/install-habitat).

As always thank you for using Chef Habitat!

Highlights:

#### Bug Fixes
- Don&#39;t panic in the unusual event persisting butterfly data takes over 30 s [#6018](https://github.com/habitat-sh/habitat/pull/6018) ([baumanj](https://github.com/baumanj))
- [hab] Avoid install failure when running concurrently [#5999](https://github.com/habitat-sh/habitat/pull/5999) ([stevendanna](https://github.com/stevendanna))

#### New Features & Enhancements
