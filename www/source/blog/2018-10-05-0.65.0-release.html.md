Habitat 0.65.0 Release notes

We are happy to announce the release of Chef Habitat 0.65.0. We have a couple of new features as well as bug fixes, so please read on for all the details. If you just want the binaries, head on over to [Install Chef Habitat](https://www.habitat.sh/docs/install-habitat/install-cli.

A couple items of special note:

* We are  releasing a new Launcher that adds protection against abandoning services after a Supervisor SIGHUP or upgrade.
* We now have a formalized HTTP API for the supervisor which we will be evolving over time.

Thanks again for using Chef Habitat!
